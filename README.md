# Word Document Alt Text Generator

A browser-based tool that uses Claude AI to automatically generate accessibility alt text for images in Microsoft Word (.docx) documents.

## How to use

1. Open the tool in any web browser
2. Upload a Word document (.docx)
3. Review, edit, approve or skip the AI-generated alt text for each image
4. Download the updated document with alt text embedded

## Notes

- No installation required — runs entirely in the browser
- Alt text is embedded in the Word file for screen reader compatibility
- Powered by the Anthropic Claude API
